代表挨拶

TOP MESSAGE

お客様の満足のために
的確かつ親切なコスト削減ソリューションを提供

ヤブシタエネシス株式会社は2014年4月4日に電気のトータルソリューションの実現のため、設立致しました。お客様の立場になり、「気づき、そして感謝と丁寧」をモットーに活動しております。

私たちはLED照明提案、空調入れ替え、電気の見える化及び自動制御、売電を軸とした電力コンサルティングを通して企業様の経営効率の改善に寄与し、お客様の実状に合わせたソリューションのさらなる実現のために、日々提案力の強化を図っております。

また、お客様にはいつも安心してご相談いただけるように、定期的なアフターフォローや効果検証に注力しております。今後もお客様、取引先、従業員、人と人との繋がりを大切にし、「出会えてよかった！」この言葉の数を増やしていけるよう努力していく所存でございます。

今後ともヤブシタエネシスに一層のご支援を賜りますようお願い申しあげます。
